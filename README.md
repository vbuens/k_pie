Estimate the percentage of infection in a leaf

scikit-learn==0.21.0

six==1.12.0
sklearn==0.0
utils==0.9.0
webcolors==1.8.1


## Usage

python perc_infection.py -o output-dir/ -i input-dir -p infected.jpg -n healthy.jpg -k 5 
-k = 5 (by default)
-o = results (by default)
